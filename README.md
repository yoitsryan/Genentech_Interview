# Genentech_Interview
Here is the code and the output files for the interview for the Analytical Data Science Programmer position at Genentech.

The folder "Question 1" has the code to clean the DS (disposition) domain from pharmaverse. This code is in the file question_1.R. The output file is DS_domain.csv.

The folder "Question 2" has the code to create an ADSL table and derive 5 new variables from multiple domain tables from pharmaverse. This code is in the file question_2.R. The output file is ADSL.csv.

The folder "Question 3" has code that deals with adverse effects (AE's). 01_create_ae_summary_table.R has the code to create a summary table of adverse effects, AE_summary_table.pdf. 02_create_visualzations.R has the code to create two visualizations: AE_barplot.pdf and top10_AE.pdf.
